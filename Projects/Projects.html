My projects

SomeBasicMathsNotions

SomeBasicMathsNotions is a small reference website I've made for Russian mathematics students and enthusiasts who are only beginning to study algebra. It explains some basic and important algebraic concepts. The source is available on my Github page. The website can be found at this location.

LlIF

LlIF (Linux loves Interactive Fiction) is a tiny script to add IF support in GNU/Linux OS (it works best for Debian-based distros). It adds appropriate story file MIME-types with icons to OS and attempts to install software from Debian repos. For more information see LlIF website.

LineCodes

Line Codes from PNG images converter is a raw and small Jupyter script made for a very specific task. Some parts are unfinished, but it does what it was designed to do. See also Wikipedia article on line codes.
