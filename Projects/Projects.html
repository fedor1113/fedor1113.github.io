SomeBasicMathsNotions

SomeBasicMathsNotions is a small reference web-site I've made for Russian mathematics students and enthusiasts who are only beginning to study algebra. It explains some basic and important algebraic concepts. The source is available on my Github page. The web-site can be found at this location.

LlIF

LlIF (Linux loves Interactive Fiction) is a tiny script to add IF support in GNU/Linux OS (it works best for Debian-based distros). It adds appropriate story file MIME-types with icons to OS and attempts to install software from Debian repos. For more information see LlIF web-site.
